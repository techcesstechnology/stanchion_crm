## ‚ö° Quick Start

1.  **Install dependencies:**
    ```bash
    npm install
    ```

2.  **Configure Environment:**
    Copy `.env.example` to `.env` and add your Firebase credentials.

3.  **Run locally:**
    ```bash
    npm run dev
    ```
